# Projeto de Microsserviços com Docker
Author Jaqueline Sandriele

## Descrição
Este projeto foi desenvolvido como parte de um desafio prático com foco em conteinerização, microsserviços e padronização de ambiente com Docker.

A proposta é demonstrar como aplicações podem ser separadas em serviços independentes, facilitando escalabilidade, manutenção, deploy e organização da infraestrutura.

## Objetivo
Replicar e evoluir um projeto base proposto no desafio, aplicando boas práticas de desenvolvimento e estruturação de aplicações em containers.
